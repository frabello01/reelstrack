Admin todo: replace Hide with Edited toggle, drop Hidden tab

Workflow simplification per user request:

1) All previously-hidden reels migrated to is_edited=true and
   is_hidden cleared, so historical hidden reels show up in the
   Edited Trello column instead of disappearing.

2) The eye/eye-off (Hide) button in each reel actions row is gone.
   Replaced by a prominent Editato toggle (checkmark icon) that
   flips is_edited on the row directly. Off = plain check, on =
   green pill with filled check.

3) The little Da editare/Editato chip that used to live inside
   AdminClipBadge is removed (was easy to miss). The clip count
   chip stays. The edit toggle is now the single, obvious action
   in the reel toolbar.

4) Trello tabs are 3 instead of 4 now: Pending, To be edited,
   Edited. Hidden tab dropped since hide is no longer a concept.

is_hidden column is preserved in the schema for safety but no
longer touched by the admin UI. The PublicTodoPage public query
still has .eq is_hidden false but post-migration that matches
every row.
