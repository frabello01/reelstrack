Creator clip uploads: Google Drive integration with resumable URL

End-to-end feature for creators to upload finished video clips for each
reel in a to-do list directly to a per-creator Google Drive folder.
Files go browser -> Drive without ever touching Render bandwidth, using
Drive's resumable session URL protocol with the agency's OAuth token.

Architecture decisions:
- Single agency OAuth (drive.file scope) for safer permissions and easier
  setup than per-creator OAuth flows
- Per to-do list toggle (creator_uploads_enabled) lets us A/B-test one
  list at a time
- Auto-done on first clip arrival (never reverses), with an independent
  is_edited flag the admin toggles after the editor finishes work
- Filename pattern: #N - <ig_url> - vK.ext for unambiguous tracking
- Click delete on Drive AND row to avoid orphaned files

Schema (supabase migrations/2026_06_10_creator_drive_uploads.sql):
- agency_settings + google_drive_refresh_token + email + connected_at
- talents + drive_folder_id + drive_folder_name
- todo_lists + creator_uploads_enabled + talent_id
- todo_list_reels + is_edited + edited_at + uploads_count
- todo_list_reel_uploads (one row per uploaded clip with version_number,
  drive_file_id, drive_view_url)

Backend (new files):
- lib/googleDrive.js - OAuth flow + token cache + Drive API client
- routes/drive.js - status, oauth/start, oauth/callback (HMAC CSRF),
  disconnect, folders list/validate
- routes/uploads.js - public init/complete/list/delete for the creator
  flow, admin endpoints for clip listing and edited toggle
- routes/todos.js - extended PATCH to accept talent_id, creator_uploads_enabled
- routes/talents.js - extended PATCH to accept drive_folder_id/name
- index.js - mount routers, whitelist public paths in auth gate, add
  permissive CORS for uploads/public

Frontend:
- api.js - new methods for drive, reel uploads (admin + public)
- SettingsPage - Connect/Disconnect Drive section with OAuth status
  flash on callback redirect
- TalentDetailPage - Drive folder picker with searchable modal
- TodoDetailPage - CreatorUploadsBlock toggle + talent picker with
  config checklist; AdminClipBadge per reel showing clip count + edited
  toggle + expandable Drive links
- PublicTodoPage - ClipUploadStrip per reel with drag/drop, XHR-based
  PUT with progress bar to Drive resumable session URL, version-aware
  delete with creator-facing list of uploaded clips
